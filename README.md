# AI Life Coach 项目

## 项目概述
这是一个通过火山方舟的DeepSeek R1 API实现的AI生活教练网站。用户可以通过与AI进行对话，获取个人成长建议和指导。

## 项目架构
项目采用简单的前后端分离架构：
- 前端：使用HTML5、CSS3和JavaScript构建用户界面
- 后端：使用Node.js创建简单的服务器，处理API请求并解决CORS问题

## 页面结构

### 主页 (index.html)
- **页眉区域**：包含网站标题和简短介绍
- **聊天区域**：
  - 聊天历史显示区：展示用户与AI的对话历史
  - 消息输入区：用户输入问题或描述需求的文本框
  - 发送按钮：提交用户消息
- **页脚区域**：包含版权信息和必要链接

### 样式说明
- 采用响应式设计，适配不同设备屏幕
- 使用柔和的色彩方案，创造舒适的对话环境
- 聊天气泡区分用户和AI消息
- 加载动画提示AI正在思考/回复

## 功能特点
- 实时对话：用户可以与AI进行实时对话
- 流式输出：AI回复采用流式输出，提供更自然的对话体验
- 上下文记忆：AI能够记住对话上下文，提供连贯的建议
- 响应式设计：在各种设备上都有良好的显示效果

## 技术实现
- 前端使用纯HTML5和CSS3，确保代码简洁易懂
- 使用Fetch API与后端通信
- 后端使用Node.js和Express框架
- 通过火山方舟的DeepSeek R1 API实现AI对话功能

## 使用说明
1. 启动后端服务器：`node server.js`
2. 在浏览器中打开index.html
3. 在聊天框中输入问题或需求
4. 点击发送按钮或按Enter键发送消息
5. 等待AI回复并继续对话
